IdiomX Dataset – English Idioms for NLP

This dataset contains ~12,800 English idioms extracted from Wiktionary (Kaikki.org) and WordNet.

It is designed for:
- Idiom detection
- Semantic understanding
- Context-to-idiom modeling
- NLP research

Columns:
- idiom_canonical
- idiom_surface
- example
- idiom_canonical_meaning
- source
- pos
- tags

Formats:
- CSV
- Parquet

Includes:
- Cleaned dataset
- Sample dataset
- Statistics

Note:
Raw data and intermediate files are excluded from this release.
