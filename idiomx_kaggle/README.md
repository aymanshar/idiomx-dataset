# 🔥 IdiomX: English–Arabic Idiom Understanding Dataset

## 📌 Overview
IdiomX is a large-scale bilingual dataset for idiomatic expression understanding.

- 123,000+ contextual examples
- 15,000+ English idioms
- English ↔ Arabic meanings
- Context-aware usage
- Semantic annotations

---

## 🧠 Tasks Supported

- Idiom Detection  
- Context → Idiom  
- Arabic → Idiom  
- Meaning Retrieval  
- Semantic Understanding

## 📊 Dataset Structure

| Column | Description |
|--------|------------|
| idiom | idiom expression |
| meaning_en | English meaning |
| meaning_ar | Arabic meaning |
| example | sentence |
| source | origin |
| pos | part of speech |
