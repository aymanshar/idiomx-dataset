# IdiomX Dataset

## Description
IdiomX is a curated dataset of English idioms extracted from Wiktionary and WordNet, designed for NLP research and idiomatic language understanding.

## Dataset Size
~12,800 idioms

## Languages
- English (primary)
- Arabic (optional enrichment)

## Fields
- idiom_canonical
- idiom_surface
- example
- idiom_canonical_meaning
- source
- pos
- tags

## Source Data
- Wiktionary (via Kaikki.org)
- WordNet (Princeton)

## License
- CC BY-SA 4.0 (Wiktionary-derived)
- WordNet License

## Intended Use
- Research
- Model training
- Semantic understanding

## Limitations
- Some examples missing
- WordNet portion is small

## Ethical Considerations
Dataset contains linguistic data only; no personal or sensitive information.
